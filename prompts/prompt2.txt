## Convenciones de nomenclatura

### Archivos (sufijos obligatorios)
| Tipo | Formato | Ejemplo |
|------|---------|---------|
| Modelo | `<nombre>_modelo.py` | `producto_modelo.py` |
| Controlador | `<nombre>_controlador.py` | `producto_controlador.py` |
| Repositorio | `<nombre>_repositorio.py` | `producto_repositorio.py` |
| Vista | `<nombre>_vista.py` | `producto_vista.py` |
| Componente | `<nombre>.py` | `boton_primario.py` |
| Test | `test_<nombre>.py` | `test_modelo.py` |

### Clases (sufijos obligatorios en mayúscula inicial)
| Tipo | Formato | Ejemplo |
|------|---------|---------|
| Modelo | `{Nombre}Modelo` | `ProductoModelo` |
| Controlador | `{Nombre}Controlador` | `ProductoControlador` |
| Repositorio | `{Nombre}Repositorio` | `ProductoRepositorio` |
| Vista | `{Nombre}Vista` | `ProductoVista` |
| Componente | `{Nombre}` (sin sufijo) | `BotonPrimario` |

### Métodos y funciones
- **Español**: `validar_nombre()`, `listar_ejemplo()`, `guardar()`, `obtener_todos()`
- **Snake case**: `crear_producto()`, `reducir_stock()`
- **Verbo + sustantivo**: `guardar_producto()`, `obtener_cliente()`

### Variables
- **Español**: `nombre`, `precio`, `stock`, `descripcion`, `producto`, `usuario`
- **Snake case**: `producto_seleccionado`, `lista_clientes`, `es_valido`

### Constantes
- **MAYÚSCULAS con guiones bajos**: `DB_PATH`, `MAX_STOCK`, `PRECIO_MINIMO`

Formato Google (recomendado)
def validar_nombre(nombre: str) -> bool:
    """
    Valida que el nombre no esté vacío.
    
    Args:
        nombre: El nombre a validar
    
    Returns:
        bool: True si el nombre es válido
    
    Raises:
        ValueError: Si el nombre está vacío o es solo espacios
    """
    if not nombre or not nombre.strip():
        raise ValueError("El nombre es obligatorio")
    return True

producto_modelo.py
python
"""
Modelo del módulo Producto
"""
class ProductoModelo:
    def __init__(self, id: Optional[int] = None, nombre: str = "", 
                 precio: float = 0.0, stock: int = 0):
        self.id = id
        self.nombre = nombre
        self.precio = precio
        self.stock = stock
        self.activo = True

    def __str__(self) -> str:
        return f"{self.nombre} (S/.{self.precio:.2f})"

    def to_dict(self) -> dict:
        return {
            "id": self.id,
            "nombre": self.nombre,
            "precio": self.precio,
            "stock": self.stock,
            "activo": self.activo,
        }
